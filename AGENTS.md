# R&K Flow Skills

R&K Flow 是一套 Spec 驱动式开发 Skills 体系，通过 Obsidian 管理文档，用 Agent Teams 多角色协作架构驱动开发流程。

## 项目身份

- **类型**: 开发工作流框架 / Skills 工具库
- **运行时**: Claude Code / Codex / compatible coding agents
- **文档系统**: Obsidian
- **版本控制**: Git

## 详细规则入口

@import .agents/rules/

## 目录路由

- `spec-*`：R&K Flow 核心 Spec 生命周期 Skills
- `exp-*`：经验与知识记忆 Skills
- `.agents/rules/`：本仓库长期规则、项目偏好和文档规范
- `spec/context/experience/`：显式经验记忆
- `spec/context/knowledge/`：显式知识记忆

> AGENTS.md 是入口清单，不承载长篇规范。详细规则和项目偏好维护在 `.agents/rules/`。
